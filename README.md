# Pomodoro Timer iOS App

一個美觀且功能完整的 iOS Pomodoro 計時器應用程式。

## 功能特點

- ✅ **標準 Pomodoro 計時**
  - 工作時間：25 分鐘
  - 短休息：5 分鐘
  - 長休息：15 分鐘（每完成 4 個 Pomodoro 後）

- ✅ **完整的計時器控制**
  - 開始/暫停
  - 重置
  - 跳過當前階段

- ✅ **美觀的 UI 設計**
  - 圓形進度條顯示
  - 漸變背景
  - 流暢的動畫效果
  - 根據工作/休息狀態改變顏色

- ✅ **狀態追蹤**
  - 顯示已完成的 Pomodoro 數量
  - 自動在工作和休息之間切換

## 技術架構

- **SwiftUI** - 現代化的 UI 框架
- **Combine** - 響應式編程
- **MVVM 架構** - 清晰的代碼結構

## 專案結構

```
PomodoroTimer/
├── PomodoroTimerApp.swift    # App 入口點
├── ContentView.swift         # 主視圖
├── PomodoroTimer.swift       # 計時器邏輯模型
└── Info.plist               # 應用配置
```

## 如何使用

### 在 Xcode 中打開專案

1. 打開 Xcode
2. 選擇 "File" > "New" > "Project"
3. 選擇 "iOS" > "App"
4. 將這些文件複製到新專案中，或直接使用現有的 SwiftUI 專案模板

### 運行應用

1. 選擇目標設備或模擬器
2. 點擊運行按鈕（⌘R）

### 使用計時器

- **開始**：點擊中央的播放按鈕開始計時
- **暫停**：點擊暫停按鈕暫停計時
- **重置**：點擊左側的重置按鈕重置當前階段
- **跳過**：點擊右側的跳過按鈕跳過當前階段

## 系統要求

- iOS 14.0 或更高版本
- Xcode 12.0 或更高版本
- Swift 5.0 或更高版本
